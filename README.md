## Details

* WM: [i3-gaps](https://github.com/Airblader/i3)
* Term: [urxvt](https://wiki.archlinux.org/index.php/Rxvt-unicode)
* Shell: zsh ([Oh-My-Zsh](http://ohmyz.sh/))
